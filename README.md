ChangeFileDirectory
window에서 파일의 경로를 복사할때 항상 '/' 자리에 '\\'가 적혀있습니다.
문제를 해결할 다른 방법들도 있지만 이 extension을 사용하면 붙여넣기를 할때 자동으로 변경해줍니다.
물론 변경하기 싫다면 기능을 꺼놓을 수도 있습니다
#alt+c -> on/off

사용방법은 간단합니다.
extension이 활성화된 상태로 경로를 붙여넣으시면 됩니다.
extension이 활성화 되어있을 때 undo(ctrl+z)가 먹히지 않는데
이때는 extension을 비활성화 하시고 undo하시면 됩니다.
